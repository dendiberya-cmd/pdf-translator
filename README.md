# PDF Translator — Технический перевод с китайского

Веб-приложение для перевода PDF-файлов с китайского языка на русский и английский с сохранением разметки, шрифтов и изображений.

## Возможности

- **Перевод ZH → RU / EN** — бесплатно, без API-ключей
- **5 движков на выбор**: Google Translate, Bing, Alibaba, Baidu, Yandex
- **Сохранение разметки** — таблицы, колонки, заголовки остаются на своих местах
- **Сохранение изображений** — логотипы, схемы, фотографии не затрагиваются
- **Поддержка PDF до 100 МБ** и документов до 100+ страниц
- **Прогресс-бар** в реальном времени
- **Drag & Drop** загрузка файлов

## Требования

- **Node.js** 18+
- **Python** 3.9+

## Установка

```bash
# 1. Перейдите в папку проекта
cd pdf-translator

# 2. Установите npm-зависимости
npm install

# 3. Установите Python-зависимости
pip install pymupdf translators

# 4. Запустите в режиме разработки
npm run dev
```

## Использование

1. Откройте http://localhost:5000 в браузере
2. Выберите язык перевода (Русский или English)
3. Выберите движок перевода (Google, Bing, Alibaba, Baidu, Yandex)
4. Загрузите PDF с китайским текстом
5. Дождитесь завершения перевода
6. Скачайте переведённый PDF

## Как работает перевод

1. **Извлечение** — PyMuPDF парсит PDF, извлекает текстовые блоки с позициями, шрифтами и цветами
2. **Фильтрация** — определяет блоки с китайскими иероглифами, пропускает числа и единицы измерения
3. **Перевод** — каждый блок переводится через выбранный движок (Google/Bing/Alibaba/Baidu/Yandex)
4. **Подстановка** — оригинальный текст закрашивается белым, переведённый вставляется на его место
5. **Адаптация** — размер шрифта автоматически уменьшается если перевод длиннее оригинала
6. **Автофоллбэк** — если один движок недоступен, автоматически пробуются альтернативные
